# Syed Areeb Ashraf — Portfolio

**CS Graduate | Systems Thinker | Builder of Intelligent Things**

A modern, full-featured portfolio website showcasing my work in machine learning, backend systems, full-stack development, and automation. Built with vanilla HTML, CSS, and JavaScript featuring custom interactions, particle animations, and responsive design.

---

## 🎨 Features

- **Custom Particle Canvas Animation** – Interactive canvas-based background with animated particles
- **Typewriter Effect** – Dynamic hero section with rotating taglines
- **Custom Cursor** – Premium tracking dot/ring cursor (desktop only)
- **Smooth Scroll Reveal** – Intersection Observer-based fade-in animations
- **Responsive Design** – Mobile-first layout with hamburger navigation
- **Dark Theme with Neon Accents** – Modern glassmorphic panels with gradient overlays
- **Dot Grid Texture** – Subtle scanline and texture overlays
- **Semantic HTML** – Clean structure with proper ARIA labels
